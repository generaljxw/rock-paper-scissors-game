## 6. 使用说明

### 6.1 新文档发布流程
1. 在对应目录创建新文档
2. 填写版本控制字段
3. 提交评审并获得签字
4. 创建基线版本
5. 更新本文档索引表

### 6.2 文档检索方法
1. **按名称检索**: 在对应文档类型表中查找
2. **按阶段检索**: 在"按项目阶段检索"表中查找
3. **按责任人检索**: 在"按责任人检索"表中查找

### 6.3 基线版本获取
1. 打开对应文档类型的 `baseline/` 目录
2. 查找所需版本的文档
3. 确认文档状态为"基线版本"
